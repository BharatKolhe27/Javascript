# JavaScript Concepts 

Revise the javaScript concepts 

##  Topics Covered

### 1. Variables & Control Flow
- `var`, `let`, `const`
- Truthy/Falsy & Comparison Operators
- Loops: `for`, `while`, `do-while`
- Switch & Ternary Operator

### 2. Arrays
- Manipulating with `push`, `pop`, `shift`, `unshift`
- The Spread Operator & Cloning
- Destructuring & Array Methods (`map`, `filter`, `reduce`, etc.)

### 3. Objects
- Dot vs Bracket Notation
- Computed Properties
- Destructuring & Spread
- Array of Objects & Nested Destructuring

### 4. Functions
- Declarations, Expressions, and Arrow Functions
- Scope (Lexical vs Block)
- Default & Rest Parameters
- High-Order Functions & Callbacks

### 5. Utilities
- Sets & Maps
- Object.assign & Optional Chaining
